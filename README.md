# Vrinda Store Excel Dashboard 📊

An interactive Excel dashboard for **Vrinda Store Annual Report 2022**, showcasing e-commerce sales performance, order trends, and customer demographics using PivotTables, charts, and slicers.

> 📌 Built entirely using **Microsoft Excel**, this project demonstrates real-world data analysis and business reporting for an online retail store.

---

## 🔍 Project Overview

This dashboard provides a comprehensive analysis of Vrinda Store's performance in 2022, including:

- **Sales by State** and **Order Status**
- **Top Performing Channels** (Amazon, Flipkart, etc.)
- **Monthly Trends**: Orders vs Sales
- **Demographic Analysis** by **Age & Gender**
- **Sales Comparison**: Women vs Men

All charts and data visuals are fully interactive using slicers for:
- Month
- Sales Channel
- Product Category

---

## 📈 Key Business Insights

- **Amazon** dominates sales with **35%** of total orders.
- **Maharashtra** and **Karnataka** lead in revenue, contributing over **5.6M** combined.
- **92% of orders were delivered**, indicating strong fulfillment efficiency.
- **Women drive 64%** of total sales, especially in the **Adult and Teenager** segments.
- Peak sales occurred from **Feb to May**, showing seasonal buying patterns.
